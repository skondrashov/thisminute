# Map Situation Labels

Rules for generating `map_label` and `registry_label` for thisminute.org events. These labels drive the map display and story-to-event matching. This document is the single source of truth — all LLM prompts must conform to these rules.

---

## map_label (what the map shows)

A 2-4 word phrase rendered at the event's geographic coordinates. The reader's eye is already on the location — the label's only job is to say **what is happening there**.

### The 10 Rules

1. **Lead with a strong verb.** Every label must contain an action verb in present tense or present participle. "Tariffs escalate", "Dam collapses", "Nurses striking". No verbless noun phrases.

2. **Never include place names.** Country, city, region names are already shown by the map marker. "Tariffs escalate" not "US-China tariffs". If removing the place name makes the label meaningless, the label is too vague.

3. **No proper nouns.** No person names, company names, organization names, party names. These belong in registry_label only. "Fleet grounded" not "Boeing grounded". "President impeached" not "Trump impeached".

4. **No filler words.** These words carry zero information and are banned:
    - crisis, situation, issues, developments, concerns, tensions, problems, unrest, turmoil, upheaval, challenges, dynamics, matters, affairs, conditions, circumstances, implications, ramifications, aspects, factors, elements

5. **No passive or observational voice.** The label describes what's happening, not how it's being perceived. "Nurses strike" not "Nurses' demands highlighted". "Bridge collapses" not "Bridge collapse reported". Never use: "receives", "publishes", "faces", "amid", "sparks debate", "draws attention", "raises questions".

6. **2-4 words maximum.** Every word costs readability at low zoom. Prefer 2-3 words. If you need 5, you're trying to say too much — split into the core action.

7. **Be concrete and specific.** Name the exact thing happening. "Cholera outbreak" not "Health emergency". "Dam collapses" not "Infrastructure failure". "Teachers protest pay" not "Education sector disputes". Go one level more specific than your first instinct.

8. **One event = one action.** Don't combine multiple things: "Protests and flooding" is two events. Pick the dominant one.

9. **Present tense for ongoing, past participle for completed.** "Ceasefire holds" (ongoing), "Minister resigns" (just happened), "Hostages released" (completed). Never future tense.

10. **Must work without any context.** A reader who has never heard of this event should understand the label at a glance. No abbreviations (no "PM", "FM", "GDP"), no jargon, no assumed knowledge.

### Quality Test

Ask: "If I saw only this label floating on a map with no other context, would I immediately know what kind of thing is happening there?" If no, rewrite.

### Real Production Labels — Rated

These were generated by the system. Learn from the good ones, avoid patterns from the bad ones.

**GOOD (keep as-is):**

- "Chief Minister steps down" — concrete action, no place name
- "Naval vessel sinks" — specific, vivid
- "Nurses strike" — minimal, clear
- "Airstrikes intensify" — strong verb, present tense
- "Ceasefire talks resume" — specific phase of action

**BAD (patterns to avoid):**

- "Infrastructure and economic developments" — two banned filler words, no verb, too long
- "Innovation advances amid debate" — "amid" is observational, "advances" is vague
- "Tournament victory marred by violence" — passive voice, too long (5 words)
- "Receives mixed reviews" — observational, no subject, passive
- "Publishes year-end reviews" — observational, routine, not an event
- "Military operations escalate" — "operations" is vague; say what: "Airstrikes escalate" or "Ground offensive begins"
- "Conflict intensifies" — which conflict? Too vague without specifics. Better: "Airstrikes intensify" or "Shelling escalates"
- "Political landscape shifts" — "landscape" and "shifts" are both vague

**DUPLICATE PATTERN to avoid:**
When 5+ events in the same region get similar labels ("Military escalates", "Conflict grows", "Strikes continue", "Attacks resume", "Fighting intensifies"), the labels become useless — they all say the same vague thing. Each label must distinguish its event: "Drone strikes resume", "Hostage talks stall", "Border clashes erupt", "Naval blockade tightens".

## registry_label (for story matching)

A 3-6 word canonical identifier that lets the LLM match incoming stories to the correct event. Includes proper nouns. Stable over the event's lifetime.

### Rules

1. **Include the key proper nouns** that uniquely identify this event: person names, organization names, specific place names.
2. **Include the event type**: strike, tariffs, earthquake, election, crash, investigation, etc.
3. **Must distinguish from similar events in the same region.** "Iran Israel missile exchange" vs "Iran nuclear talks" — both involve Iran but are different events.
4. **Stable over time.** Don't rename as the story evolves. "Boeing 737 MAX grounding" stays the same even when flights resume — the event is still about that grounding.
5. **Not a headline.** No dates, no numbers, no articles (a/an/the). "Trump China tariffs" not "Trump announces 25% tariffs on March 4".

## When Labels Are Generated

Labels are created in three contexts. The rules above apply to ALL of them.

### 1. Story Extraction (new event proposal)

When the LLM extracts a story that doesn't match any existing registry event, it proposes both labels in the `new_event` field. This is the most common path for label creation.

### 2. Seed/Backfill Script

Bulk conversion of existing events into registry entries. Uses the same rules but processes many events at once.

### 3. Registry Maintenance (relabeling)

Periodic LLM review that fixes bad labels, merges duplicates, and improves map_labels. The maintenance LLM should apply these rules when proposing new map_labels.

## Anti-Patterns Checklist

Before accepting any map_label, verify it passes ALL of these:

- [ ] Contains a verb? (reject if no)
- [ ] Free of place names? (reject if any country/city/region name)
- [ ] Free of proper nouns? (reject if person/company/org name)
- [ ] Free of banned filler words? (reject if crisis/situation/issues/developments/concerns/etc.)
- [ ] 2-4 words? (reject if 5+)
- [ ] Not passive/observational? (reject if "receives", "faces", "amid", "draws", "raises")
- [ ] Specific enough to distinguish from similar events? (reject if could apply to 5+ different events)
- [ ] Understandable without context? (reject if requires prior knowledge)
